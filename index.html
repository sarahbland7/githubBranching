Commands

Clone


							git clone https://github.com/rohitmusti/teaching-github.git

Literally creates a "clone" of the repository on the user's local computer

Add


							git add .
						

							git add filename

Allows for fine control of which files should be added to the group to be saved

Commit


							git commit -a -m "Message Here"

The -a flag does the job of the previous add instruction

The -m flag allows for the version being "committed" to have a message stored with it
